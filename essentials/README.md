# Go Language Tutorial

Welcome to this comprehensive Go language tutorial. Go is a statically typed, compiled language designed at Google for efficiency and scalability.

## Getting Started

### 1. Installation
Ensure you have Go installed on your system. You can download it from [golang.org](https://golang.org/dl/).

### 2. Project Structure
This tutorial is organized into separate files to demonstrate different Go concepts.

## Tutorial Sections

### Hello World
The simplest Go program. See `hello.go`.
Run it with: `go run hello.go`

### Variables and Types
Learning about basic data types and variable declarations. See `variables.go`.
Run it with: `go run variables.go`

### Control Flow
Using if/else, for loops, and switch statements. See `control_flow.go`.
Run it with: `go run control_flow.go`

### Structs and Interfaces
Understanding Go's approach to object-oriented programming. See `structs_interfaces.go`.
Run it with: `go run structs_interfaces.go`

### Concurrency (Goroutines & Channels)
The power of Go: running functions concurrently. See `concurrency.go`.
Run it with: `go run concurrency.go`

## How to run the examples
Each file is a standalone `main` package. You can run any file using:
```bash
go run <filename>.go
```
